# Proyecto Individual Robert Melendez C34772
Este proyecto analiza y compara el comportamiento del mercado de fichajes en la temporada 2024–2025 para tres de las ligas más importantes del mundo: LaLiga, Premier League y Bundesliga.
A partir de datos obtenidos con worldfootballR y procesados en R, se estudian tendencias de gasto, nacionalidades predominantes, posiciones más cotizadas y la relación entre inversión y rendimiento deportivo.
## Objetivo
Analizar las estrategias de fichaje de las tres principales ligas europeas durante la temporada 2024-2025, identificando las diferencias en gasto total, nacionalidades predominantes, posiciones más cotizadas y la relación entre su rendimiento en la temporada anterior y su valor de transferencia.
